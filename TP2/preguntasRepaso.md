# Preguntas de repaso:

1. En cada una de las siguientes afirmaciones indique la respuesta correcta:
   ✔ Los métodos y atributos estáticos se almacenan en una zona de la memoria denominada:
   ▪ Heap
   ▪ Static --> Respuesta correcta
   ▪ Stack

   ✔ Las instancias u objetos creados a partir de una clase se almacenan en la zona de memoria:
   ▪ Stack
   ▪ Heap --> Respuesta correcta

   ✔ Un bloque de código se encuentra delimitado por los símbolos:
   ▪ ( )
   ▪ { } --> Respuesta correcta
   ▪ [ ]

   ✔ Una variable puede ser referenciada en algún lugar del programa mediante:
   ▪ Su nombre --> Respuesta correcta
   ▪ Su ubicación en la memoria
   ▪ Su tipo primitivo

2. Indique verdadero / falso según corresponda:
   ✔ Java es un lenguaje fuertemente tipado, ya que es necesario declarar el tipo de las variables a utilizar. --> V

   ✔ El tipo de dato “double” permite almacenar valores con una precisión de hasta 64 bits. --> V (64 bits = 8 bytes)

   ✔ Se dice que java no es un lenguaje “case sensitive” (sensible al uso de minúsculas y mayúsculas). --> F

   ✔ A una variable de tipo “float” se la puede inicializar con un literal de tipo flotante seguido con la letra “f” o “F”, de lo contrario se producirá un error porque se tratará al literal como “double”. --> V
